# Satellite Constellation System

## Overview

This project is a full-stack simulation system for monitoring satellites, detecting collision risks, and managing maneuvers in real time.

It consists of:

* Backend (FastAPI)
* Frontend (React Dashboard)

---

## Project Structure

```
satellite-project/
│── backend/
│── frontend/
```

---

## Features

* Real-time satellite simulation
* Collision detection system
* Maneuver scheduling & execution
* Interactive dashboard visualization
* System health & metrics tracking

---

## Tech Stack

### Backend

* FastAPI
* Python
* Pydantic

### Frontend

* React
* Tailwind CSS
* Axios

---

## How to Run

### Backend

```
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```
cd frontend
npm install
npm run dev
```
